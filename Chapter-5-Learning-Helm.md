### About Helm
Helm is a cloud package manager, in the same way you sudo yum install an rpm to your linux box/vm, you helm install something to your cloud.

Using this cloud package manager also makes the configuration of software really easy to manage, simplify Kubernetes yaml and allowing the package createtor to decide which configuration variables are useful, and which ones are not.
